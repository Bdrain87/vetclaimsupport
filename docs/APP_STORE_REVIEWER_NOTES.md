# App Store Reviewer Notes

**Category:** Education / Organization

---

**Note to Reviewer:** This app is a local-first organizational tool for Veterans to track disability claim evidence.

**Biometrics:** The app uses WebAuthn/FaceID for the "Evidence Vault." If the reviewer device does not have biometrics enabled, the app gracefully falls back to a standard secure state.

**No Data Collection:** This app does not use a remote server for user data. All symptom logs and ratings are stored exclusively on the user's device.

**AI Translation:** The "VA-Speak" feature utilizes the Google Gemini API for educational terminology mapping. No PII is stored during this process.

**Education Category:** This tool provides educational mapping to 38 CFR Part 4 (VASRD) criteria and does not provide medical diagnoses.

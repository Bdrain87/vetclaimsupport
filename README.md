# Vet Claim Support

Track and organize your VA disability claim evidence with confidence.

## About

Vet Claim Support is a Progressive Web App (PWA) designed to help veterans document health issues, track symptoms, and build strong evidence for VA disability claims. Perfect for BDD claims or documenting conditions for future claims.

## Features

- **Health Tracking**: Symptom journal, migraine log, sleep tracker, medication log
- **Evidence Building**: Document upload, evidence library, buddy statements, timeline builder
- **Claim Tools**: C&P exam prep, documents checklist, service history, 4-phase progress tracking
- **VA Conditions Reference**: Complete 38 CFR Part 4 reference database

## Privacy

Your data stays on your device. We prioritize privacy and security.

## Development

This project uses:
- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (for optional cloud features)

### Local Development

```bash
npm install
npm run dev
```
